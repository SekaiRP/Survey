به طور کلی آرپی سکای رو چطور ارزیابی می‌کنید؟ *
دلیل شما برای ارزیابی *
جو گپ سکای رو چطور ارزیابی میکنید *
دلیل شما برای ارزیابی جو (اختیاری)
تیم ادمین های سکای رو چطور ارزیابی می‌کنید؟ *
دلیل شما برای ارزیابی ادمین ها *
به نظر شما مهم ترین بخش آرپی کدومه؟ *
دلیل شما برای انتخاب بخش (اختیاری)
اگر میتونستید یه بخش به سکای اضافه کنید، چی اضافه می‌کردید؟ (اختیاری)
برای بهبود آرپی چه پیشنهادی رو ارائه میدید *
دلیل شما برای انتخاب سکای *
تا چه حد از سرویس های بخش Dev (ربات ها و...) رضایت دارید؟ *
دوست دارید چه چیز هایی به بخش Dev سکای اضافه بشه؟ (اختیاری)
